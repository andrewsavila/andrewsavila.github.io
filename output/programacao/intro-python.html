Vamos ver um exemplo básico:

def saudacao(nome):
    print(f"Olá, {nome}! Seja bem-vindo ao Pelican.")

saudacao("Andre")
Para que serve o Python

Durante muito tempo o trabalho com dados se restringia a dados que estavam em grandes bancos de dados tradicionais como Oracle, Sql Server.... com o tempo outros bancos como MySql e PostGree... dentre outros foram tomando espaço.

Porem ocorreu um fato interessante, com a advento do BigData, o DataLake inicialmente e DeltaLake na sequẽncia foram tomando para si uma responsabilidade que retirou dos bancos de dados relacionais os dados oficiais da empresa.

Conclui que alem de dominar a linguagem estruturada Sql, passou a ser importante dominar outra forma de manipular dados que não fosse estruturado. No mercado existem muitas ferramentas robusta e pagas, porem a liberdade de manipular os dados com velocidade independentemente de ferramentas de terceiro, na minha visão apareceu o python como uma ótima estratégia.

Mas eu preciso virar um programador Python, para tratar dados?

Conhecimento não ocupa espaço, uma vez que consigamos sintetiza-lo e ter mecanimos rápidos de acesso para recordar pontos chaves para resolver um problema. Com o tempo essa essencia de conhecimentos nos permite resolver qualquer problema, sem necessitar de uma memória de elefante, e teremos consciência do mecanismo utilizado para resolver esse ou aquele problema, nao delegando 100% da solução para uma IA. Isso tem um grande benefício, pois vou treinando minha mente para pensar na solução, usando a IA para complementar aquilo que não sou capaz de fazer ou simplesmente não vale a pena fazer, pelo fato da IA trabalhar melhor do que eu.

Objetivo desse site

Hoje em dia os modelos de IA, que nos responde baseado em Prompt é a melhor forma de aprender, logo surigo que procure essas ferramentas para aprofundar e entender as tecnologias que precisa. Esse site tem o objetivo, de unir de forma sintética inicialmente, os conhecimentos necessarios para manipulação de dados ,justificando o tema PlolyglotData. E cumpri outro objetivo que é pare eu aprender tambem, ao se escrever o que se aprendeu com exemplos, como se tivesse ensinando a alguem alguma coisa, o conhecimento sedimento de forma mais clara e real dentro de nossa inteligência. E caso seja útil para alguem, esse meu trabalho de síntese, seja bem vindo.
