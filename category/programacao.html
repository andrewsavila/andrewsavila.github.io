Bem-vindo ao meu site!

Ola leitor, escrevi esse site para mim mesmo, meu objetivo principal foi criar um mecanismo para registrar os estudos que faço na área de dados. São tantos aspectos, tantos conceitos que para o conhecimento ir sedimentando paulatinamente fui estruturando o estudado para cumprir basicamente com dois objetivos:

  • O primeiro é organizar os elementos estudados no passado, para que eu os vincule com os estudos recentes, quantas vezes ja vivi a situação: Já li isso uma vez... onde esta mesmo, e la vai iniciar uma nova pesquisa, se a primeira vez que estudo registro de forma organizada, no futuro tenho so que pegar o ponto de partida e complementar o que ja iniciei no passado
  • O conhecimento até ficar disponível na memoria ativa, precisa ser muitas vezes usado até o ponto de ser automatico, porem o volume de informações e conhecimentos é muito grande, dai estou usando essa técnica facilitar esse trabalho de sedimentação do que estudei

O segundo objetivo é compartilhar o que ja estudei, resolvi utilizar o Pelican, ferramenta muito bacana e útil que alem de ajudar-me a documentar o que fui aprendendo, posso ir compartilhando esse meu esforço, com quem ache útil. Escolhi essa ferramenta dentre as milhares de opções existentes, pois consigo fazer tudo localmente usando o que tem de mais simples, no caso escolhi o MarkDown, e ainda posso publicar em um site estático, tambem estou usando o GitHub na modalidade de "User Site", para configurar um dominio próprio e usar uma opção gratuita de armazenamento de um conteúdo estático. Disponibilizado gentilmente pelo GitHub

E para completar uma explicação sobre esse trabalho, ele não irá substituir o Google e os diversos mecanismos de IA Generativa( chatgpt.com, gemini.google.com e etc) para pesquisar um conteúdo de forma simples e personalizada. Esse site segue um fluxo organizado para registro de um conteúdo estruturado no tratamento de dados, e meu plano é mante-lo atualizado e agregando novos conteudos a medida que consiga estrutura-los de forma simples e clara.
